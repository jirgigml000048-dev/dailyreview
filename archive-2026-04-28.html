头条 · AI 格局重塑

微软 OpenAI 七年独家 CP 正式分手：IP 非独家至 2032、不再支付收入分成、可上 AWS

4 月 28 日，OpenAI 与微软共同宣布修订 2019 年签署的合作协议——这是双方关系第三次也是最大幅度的一次重构。三条核心变化：（1）取消微软独家销售权，OpenAI 模型可同步分发到 AWS、Google Cloud 等其他云平台；（2）IP 授权由独家改为非独家、有效期延伸至 2032 年，OpenAI 获得对外授权技术与建立其它战略合作的空间；（3）微软不再向 OpenAI 支付收入分成，但 OpenAI 继续向微软支付分成至 2030 年（比例不变、设总额上限）。Azure 仍是 OpenAI 的"首选云"，新产品仍在 Azure 首发，但"云独家"时代结束。

背后逻辑：今年 1 月 OpenAI 宣布与亚马逊达成 380 亿美元算力合作，但旧协议下微软拥有"优先购买权"和"独家分销权"——这两条本应让 OpenAI 与 AWS 的合作流产。新协议化解了这个死结。Evercore 分析指出，新协议"移除了 AGI 模糊性，明确 2032 年前微软的 IP 权限"，是双赢条款。但 OpenAI CRO Denise Dresser 上周内部备忘录刚刚指控 Anthropic"虚报 ARR"，本周又有这种结构性松绑动作——硅谷 AI 三巨头的"竞合"边界正在被重画。

对 PR 端最直接的影响——"选哪家 AI"决策正式从"选哪家云"中剥离出来。过去一年里"我用 Azure 因为我用 OpenAI"是合理的捆绑叙事，今后"我用 OpenAI 但跑在 AWS 上"也是合理叙事。企业 IR / PR 在财报、招股书、官网中提及"AI 合作方"与"云合作方"时，应分两条独立 messaging 来组织，避免投资人/媒体对"绑定深度"做错误推断。

头条 · 资本管制

Meta 20 亿美元收购 Manus 被发改委紧急叫停：技术、数据、架构三条红线全踩

4 月 27 日深夜，国家发改委外商投资安全审查工作机制办公室发布公告，对 Meta 收购中国 AI 公司 Manus 项目作出禁止决定，并要求交易当事人撤销该收购交易。这是 Meta 公司史上第三大并购案被叫停，也是 2025 年外商投资安全审查办公室成立以来作出的最严格审查结论之一。

三条核心红线：（1）技术与数据安全：Manus 核心代码、模型、Agent 架构、Workflow 与训练数据均由中国籍团队在境内研发；（2）外资并购合规漏洞：交易未履行中国外商投资安全审查程序；（3）架构重组规避监管：通过"境内研发 → 境外换壳 → 外资收购"路径转移控制权。撤销交易最棘手之处在于"知识无法物理回收"——一旦 Meta 工程师看过源代码、加入内部团队、把能力用于产品设计，证明"已完全清除收购影响"几乎不可能。20 亿美元退回流程亦尚无先例。观察者网

对 PR 端含义：（a）今后中国 AI 公司"卖身海外巨头"路径基本被堵死，独立上市 + 国资 / 政府引导基金路线成为主流退出方式；（b）涉及 AI / 数据 / 算力 / 跨境云的并购传闻在 PR 沟通中要格外谨慎，传闻本身就可能触发监管侧"提前关注"；（c）对仍在做 VIE / 红筹架构调整的次新股，要重新评估"架构透明度"对资本市场叙事的权重。

英伟达再破纪录：股价 +4% 报 $216.61，市值升破 $5.26 万亿，时隔半年重夺全球第一

4 月 28 日（北京时间），英伟达盘中刷新历史纪录，股价收涨 4%、报 216.61 美元，推动市值突破 5.26 万亿美元，是其自 2025 年 10 月底以来时隔近 6 个月重新站上 5 万亿美元大关。同步利好的还有美光 +5%、闪迪 +8%（HBM 主线）。AMD 则跌超 3%，反映出"AI 算力赢者通吃"格局被市场再次确认。

本轮反弹的驱动来自三个层面：（1）OpenAI 与微软新协议解除"云独家"，AWS、Oracle、CoreWeave 等英伟达 GPU 主要采购方将有机会承接更多 OpenAI workload，等于打开了 GPU 增量需求；（2）IDC 数据显示 2024 全球 AI 算力市场规模 3500 亿美元、预计 2027 突破 1.2 万亿美元，英伟达凭 CUDA 生态占据高端 GPU 80%+ 份额；（3）今日 A 股算力硬件板块同步爆发——东山精密、长光华芯涨停创历史新高，工业富联触及涨停，永鼎股份、杭电股份封涨停，"AI 算力"已从单一美股叙事升级为全球同步的资本市场主线。36氪
