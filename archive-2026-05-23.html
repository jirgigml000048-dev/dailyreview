头条 · AI 格局

企业采用率反超：Anthropic 工作场所占比 34.4% 历史性超越 OpenAI 32.3%

本周最具结构性意义的 AI 新闻：Anthropic 在企业工作场所的采用率达到 34.4%，首次超越 OpenAI 的 32.3%。过去一年里 Anthropic 的企业渗透率翻了近 4 倍，而 OpenAI 同期仅 +0.3%。配合本周更新的 Claude Haiku 4.5、Claude Sonnet 4.5 Extended Thinking，以及自研产品 Claude Code + Cowork 在编程市场拿下 54% 份额、企业 API 拿下 40% 份额，"谁是 AI 企业默认选项"的叙事已经从"非此即彼"转向"Anthropic 默认 + OpenAI 备选"。

与此同时 OpenAI 在 4 月底推出 GPT-5.1 / GPT-5.1-Thinking，并将产品重心从"对话交互"转向"自主智能体"——但企业转化数据并未跟上。三巨头另一边联手推出"反对抗性蒸馏联盟"，对中国大模型生态的下游"开源蒸馏路线"形成隐形封锁。澎湃 · 三巨头结盟

对 PR 端最直接的含义——2026 年 Q2 起，企业披露"AI 合作方"已从工程信息升级为品牌信号，类似 2015 年云厂商选择。客户官网、招股书、财报电话会涉及 AI 表述时，建议同时备好"主合作方 + 备选方"的双口径话术，避免下半年改口的合规风险。
